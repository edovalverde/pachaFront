la etiqueta "main" se usa comúnmente en HTML5 como una forma de marcar el contenido principal de una página. La etiqueta "main" se utiliza para indicar al navegador que el contenido dentro de ella es el contenido principal de la página.

la etiqueta "aside" se utiliza para indicar contenido que está relacionado pero que no es esencial para el flujo principal del contenido de la página. La etiqueta "aside" se utiliza para agrupar contenido que se puede considerar como complementario al contenido principal de la página.
